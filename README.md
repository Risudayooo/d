# DiscGun (working title)

銃型CDプレイヤーにディスク(曲)を差し込み、その曲の個性を戦闘能力にして戦う **ローグライク・スピードシューター** の Minecraft MOD。

- 企画仕様: [`docs/design.md`](docs/design.md)
- 未確定事項の技術検討: [`docs/open-questions.md`](docs/open-questions.md)

> ⚠️ これは **Phase 1 (MVP) のスキャフォールド** です。コアループ(銃 + ディスク + 能力インターフェース)とスピードシューター要素(ブリンク / 二段ジャンプ / スライド / パリィ)の骨格を実装しています。ローグライクのダンジョン生成・部屋・強化レイヤー・GeckoLib 腕・BGM ループ・テクスチャは **未実装(Phase 2 / TODO)**。

## 動作環境

| 項目 | バージョン |
|---|---|
| Minecraft | 1.21.1 |
| Mod Loader | Fabric Loader 0.16+ |
| 依存 | Fabric API |
| Java | 21 |

## ビルド

```bash
# 初回のみ Gradle wrapper の jar を生成(本リポジトリには同梱していません)
gradle wrapper --gradle-version 8.10

# ビルド / 開発クライアント起動
./gradlew build
./gradlew runClient
```

ビルド成果物は `build/libs/discgun-<version>.jar`。

> このスキャフォールドは依存(MC / Yarn / Fabric)をダウンロードできる環境で `./gradlew build` を通してコンパイル検証してください(作成環境ではネットワーク制約のため未検証)。

## 実装済み(Phase 1)

- **銃型CDプレイヤー**(`gun_cd_player`)+ ディスク3種(`disc_house` / `disc_dubstep` / `disc_dnb`)
- **能力インターフェース** `DiscAbility` — 各ディスクが「左クリック挙動・右クリック挙動・クールダウン」を保持
  - House: 安定連射 / 右クリックで短距離ブリンク
  - Dubstep: 高威力低速 / 右クリックで一時シールド
  - DnB: 高速低威力 / 右クリックで範囲バースト
- **銃の手触り**: マガジン式の**残弾数**+リロード(R / 空打ちで自動)、**弾道トレーサー**表示、**画面が跳ねる反動**、ディスクごとの**拡散**、画面右下の**残弾HUD**
- **データコンポーネント** で装填ディスク + 残弾を銃アイテムに保持(クライアント同期付き)
- **パルクール・ムーブメント**(Parcool! / Redliner 系のモメンタム移動 — `ParkourController`):
  - **スライド**: スプリント中にしゃがみ → 加速して滑る。勢いを維持し、スライド中にジャンプで速度を保ったまま飛ぶ(スライドホップ)
  - **ウォールラン**: 壁の横を走ると重力を殺して壁沿いに加速
  - **ウォールジャンプ**: ウォールラン中/壁際でジャンプ → 壁を蹴って跳ぶ(連続で壁を渡れる)
  - **オートボルト**: 走って段差にぶつかると自動で乗り越え
  - **二段ジャンプ**(前方ブースト付)/ **空中ストレイフ**(空中で曲がれる・速度が乗る)
  - ブリンク(左Alt)/ 銃所持中は常時移動速度+20%
- **銃の手触り**: 残弾・リロード・弾道・反動・拡散・残弾HUD(上記)
- **操作オーバーライド**: 銃所持中はバニラの Attack/Use をミックスインで抑制
- サーバ権威の射撃判定(ヒットスキャン)・クールダウン管理

## 操作

| 操作 | キー |
|---|---|
| 射撃 | 左クリック(長押しで連射) |
| ディスク固有アクション | 右クリック |
| ブリンク | 左Alt(設計上はダッシュ/スプリントキーに割当推奨) |
| 二段ジャンプ | 空中でスペース |
| スライド | スプリント中にしゃがみ(滑走中ジャンプでスライドホップ) |
| ウォールラン | 壁に沿って空中で前進(自動発動) |
| ウォールジャンプ | ウォールラン中/壁際でスペース |
| ボルト(乗り越え) | 段差に向かって前進(自動発動) |
| パリィ | F |
| リロード | R |

> ⚠️ パリィの F はバニラの「オフハンドと持ち替え」と競合します。バニラ側のキー割当を外してください(Phase 2 でミックスインによる無効化を予定)。
> ディスク装填は Phase 1 のテスト用にディスクを右クリックでインベントリ内の銃へロードします(本来はジュークボックス交換スポット = Phase 2)。

## 既知の制限 / TODO

- テクスチャ未同梱(アイテムは missing texture 表示)
- BGM ループ再生は未実装(右クリック/装填時の効果音のみ)
- パルクールは全てクライアント側の速度操作。シングルプレイ/CurseForge では快適だが、マルチサーバーでは "moved too quickly" 等で弾かれる可能性あり(Phase 2 でサーバ側補正を検討)
- スライドはまだヒットボックス/カメラ高さの変更なし(Phase 2 でミックスイン対応)
- ローグライク部屋生成・パーク/スキル/武器MOD・ショップ等は Phase 2
